# GPS Navigation System using Graph

## Team Members
- P.Sasidhar
- VSB.Varun

## Problem Statement
To design a GPS navigation system that finds the shortest path between locations using a graph data structure.

## Data Structure Used
Graph (Adjacency List)

## Algorithm Used
Dijkstra's Algorithm

## Features
- Add Location
- Add Road
- Delete Location
- Delete Road
- Update Location
- Update Road
- Display Map
- Find Shortest Path

## How to Run
gcc src/main.c -o project
./project
